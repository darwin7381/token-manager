## 🎨 主題系統

### Light Theme (預設)
- 背景: 白色/淺灰
- 文字: 深色
- 卡片: 白色
- 陰影: 柔和

### Dark Theme (點擊🌙切換)
- 背景: 深藍/黑色
- 文字: 淺色
- 卡片: 深色
- 陰影: 加深

**CSS 變數系統**:
```css
:root {
  --bg-primary: #ffffff;
  --text-primary: #0f172a;
  --accent-primary: #3b82f6;
  ...
}

[data-theme='dark'] {
  --bg-primary: #0f172a;
  --text-primary: #f1f5f9;
  ...
}
```

**切換邏輯**:
- 點擊 🌙 → 變成 ☀️
- 自動應用到所有組件
- 平滑過渡動畫
